== Create DNS record

=== Cloudflare

----
curl https://api.cloudflare.com/client/v4/zones/$ZONE_ID/dns_records \
    -H 'Content-Type: application/json' \
    -H "X-Auth-Email: $CLOUDFLARE_EMAIL" \
    -H "X-Auth-Key: $CLOUDFLARE_API_KEY" \
    -d '{
          "name": "example.com",
          "ttl": 3600,
          "type": "A",
          "comment": "Domain verification record",
          "content": "198.51.100.4",
          "proxied": true
        }'
----

== Record Types

=== Cloudflare

 A NS SOA HINFO MX TXT AAAA LOC SRV NAPTR CERT SSHFP RRSIG NSEC DNSKEY
 TLSA SMIMEA HIP CDS CDNSKEY OPENPGPKEY SVCB HTTPS URI CAA PTR


=== deSEC

 At least the following record types are supported: A, AAAA, AFSDB,
 APL, CAA, CDNSKEY, CDS, CERT, CNAME, DHCID, DNAME, DNSKEY, DLV, DS,
 EUI48, EUI64, HINFO, HTTPS, KX, L32, L64, LOC, LP, MX, NAPTR, NID, NS,
 OPENPGPKEY, PTR, RP, SMIMEA, SPF, SRV, SSHFP, SVCB, TLSA, TXT, URI.
 (The SOA record is managed automatically.)


=== IONOS

 A, AAAA, CNAME, MX, NS, SOA, SRV, TXT, CAA, TLSA, SMIMEA, SSHFP, DS,
 HTTPS, SVCB, CERT, URI, RP, LOC, OPENPGPKEY


=== Google

 A AAAA ALIAS CAA CNAME DNSKEY DS HTTPS IPSECKEY MX NAPTR NS PTR SOA
 SPF SRV SSHFP SVCB TLSA TXT


=== Azure

 A AAAA CAA CNAME MX NS PTR SOA SRV TXT


=== GoDaddy

 A, AAAA, CNAME, MX, NS, SOA, SRV, TXT


=== DNSmadeEasy

 A, AAAA, ANAME, CNAME, HTTPRED, MX, NS, PTR, SRV, TXT, SPF, or SOA
