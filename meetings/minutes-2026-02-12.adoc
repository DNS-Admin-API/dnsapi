== API draft

Carsten offered to write a draft specification for the DNS Admin API early
January, but he didn't manage mostly due to the complexity that comes with
writing an API that uses the OpenAPI schema.

Thanks to Pawel, who pointed him towards https://typespec.io, he will now give
it a second shot using the typespec client - a tool to design "data up front and
generate schemas, API specifications, client / server code, docs, and more. "

We agreed Carsten will present the API draft in our next meeting.
